# Procedimientos Almacenados

Implementacion de la consulta de procedimientos almacenados

## Rutas
Se deben implementar las siguientes rutas:
```
matrixWebServer
│   README.md
│
├───controllers
│   │
│   └───procedures
│       │
│       ├   getAseguradorasByDoc.ts
│       └   ...
│
├───docs
|   |
|   ├───procedures
|   |   ├    getAseguradorasByDoc.md
|   |   └    ...
|   |
|   └   Procedures.md
|
└───routes
    ├───procedures
    |   │
    |   ├   proceduresRoutes.ts
    |   └   ...
    |
    └   index.ts
```
